Purrfect Relaxation

Purrfect Relaxation was my first coding project ever and the practical exercise for the SheCodes Basic Introduction to Coding workshop, I passed on 11 September 2021. The project included basic HTML, CSS, JavaScript and VS Code. The task was to create a landing page from scratch about something we were passionate about, including user interaction. As I love (my) cats, it was clear to build a page about/around them.

Talk about the Weather

Talk about the Weather was the practical exercise for the SheCodes Plus workshop including advanced HTML, CSS, JavaScript, VS Code, APIs, Github, Bootstrap and hosting. This workshop I successfully completed on 15 November 2021. The app runs with real live data by API and has, only for special entertainment purposes - as I find weather sometimes quite dreary - an array (pun intended) of classic poems which are being shown randomly by pressing a button if the user of the app so chooses.
